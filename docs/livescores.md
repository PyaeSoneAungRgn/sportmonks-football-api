# Livescores

## GET Inplay Livescores

```php
use SportmonksFootballApi;

SportmonksFootballApi::livescore()->inplay();
```

## GET All Livescores

```php
use SportmonksFootballApi;

SportmonksFootballApi::livescore()->all();
```

## GET Latest Updated Livescores

```php
use SportmonksFootballApi;

SportmonksFootballApi::livescore()->lastUpdated();
```
