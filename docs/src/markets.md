# Markets

## GET All Markets

```php
use SportmonksFootballApi;

SportmonksFootballApi::market()->all();
```

## GET Market by ID

```php
use SportmonksFootballApi;

SportmonksFootballApi::market()->byId($id);
```

## GET Market by Search

```php
use SportmonksFootballApi;

SportmonksFootballApi::market()->search($name);
```
