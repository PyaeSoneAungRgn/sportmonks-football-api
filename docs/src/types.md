# Types

## GET All Types

```php
use SportmonksFootballApi;

SportmonksFootballApi::type()->all();
```

## GET Type by ID

```php
use SportmonksFootballApi;

SportmonksFootballApi::type()->byId($id);
```

## GET Type by Entity

```php
use SportmonksFootballApi;

SportmonksFootballApi::type()->byEntity();
```
